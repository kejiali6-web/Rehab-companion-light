Week 2 ‚Äì Prototype Development

Progress Overview

This week, I met with my EGR 102 professor to discuss the technical architecture of Rehab Companion Light and refined the prototype concept. With his guidance, I built the first functional diagram using Arduino Uno as the main controller.

  • Voice Feedback ‚Üí Active buzzer for encouragement & alerts.
  • Light Feedback ‚Üí Green LED (on when target pressure met) / Red LED (on when incorrect pressure detected).
  • Sensor Input ‚Üí Pressure sensor and potentiometer to detect applied force.
  • Difficulty Adjustment ‚Üí Adjustable via sensor thresholds to match recovery stages.

I also learned how to use the Arduino serial monitor as a counter to track successful pressure events. Once a target threshold is met multiple times, the system triggers the buzzer as verbal encouragement. In the future, the buzzer will be replaced by a recorded voice module for more human-centered interaction.

Prototype Structure

The early-stage prototype is designed around Arduino Uno as the main controller. It integrates real-time pressure feedback, LED indicators, and adaptive difficulty settings to motivate patients through progressive rehabilitation stages.

  • Voice: Active buzzer to simulate verbal encouragement.
  • Light: Dual LED feedback for success/failure recognition.
  • Sensor: Pressure sensor and potentiometer for input sensing.
  • Difficulty: Dynamic adjustment via code-controlled thresholds.

The schematic was hand-drawn during a mentoring session, where my professor kindly lent me his personal sensor. This collaboration helped transform the idea into a tangible model ready for digital simulation.

Arduino Logic Example


int pressurePin = A0;
int greenLED = 8;
int redLED = 9;
int buzzer = 10;
int threshold = 500;
int count = 0;

void loop() {
  int pressure = analogRead(pressurePin);
  if (pressure > threshold) {
    digitalWrite(greenLED, HIGH);
    digitalWrite(redLED, LOW);
    count++;
    if (count % 5 == 0) tone(buzzer, 1000, 200);
  } else {
    digitalWrite(redLED, HIGH);
    digitalWrite(greenLED, LOW);
  }
}
