Week 2 ‚Äì Prototype Development

This week, I met with my EGR 102 professor to refine the prototype‚Äôs structure. Together, we mapped out a system using Arduino Uno as the controller, focusing on feedback through light, sound, and pressure sensors.

  • Voice Feedback ‚Üí Active buzzer for encouragement & alerts.
  • Light Feedback ‚Üí Green LED (target met) / Red LED (needs correction).
  • Sensor Input ‚Üí Pressure sensor + potentiometer.
  • Difficulty Adjustment ‚Üí Threshold-based scaling for recovery stages.

The prototype promotes effort recognition ‚Äî not just movement detection. It rewards correct pressure with light and tone, helping stroke patients feel progress in real time.

  • Controller: Arduino Uno
  • Inputs: Pressure Sensor & Potentiometer
  • Outputs: Green/Red LEDs, Buzzer

Arduino Logic Example


int pressurePin = A0;
int greenLED = 8;
int redLED = 9;
int buzzer = 10;
int threshold = 500;
int count = 0;

void setup() {
  pinMode(greenLED, OUTPUT);
  pinMode(redLED, OUTPUT);
  pinMode(buzzer, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int pressure = analogRead(pressurePin);
  if (pressure > threshold) {
    digitalWrite(greenLED, HIGH);
    digitalWrite(redLED, LOW);
    count++;
    if (count % 5 == 0) tone(buzzer, 1000, 200);
  } else {
    digitalWrite(redLED, HIGH);
    digitalWrite(greenLED, LOW);
  }
}
